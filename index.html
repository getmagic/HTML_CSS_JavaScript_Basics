HTML, CSS, and JavaScript Basics

Introduction

HTML, CSS, and JavaScript are all needed to make a website. HTML stands for Hypertext Markup Language. It is the content and core of the website. This language practices a variety of headings, paragraphs, pictures, videos. CSS stands for Cascading Style Sheets. This language adds styling to the website and makes the website look more authentic. JavaScript allows interactivity and adds behavior. The official name of JavaScript is ECMAScript.



HTML

Hypertext Markup Language

HTML is the first step to making a website. This programming language is made up of many elements. These elements have opening and closing tags, along with content inside of the tags. Most syntax in html has "<____>" for opening tags, but for closing tags, the syntax usually has "/<____>". For some syntax, such as "img", there is no need for "" as the code will only need an opening tag of "" and a closing tag of "/>".

Here is the list of basic commands in HTML

  1. Doctype: Defines a document
  2. html: Specifies which version of html you are using
  3. head: Defines information about the document
  4. title: Defines a title for the document
  5. body: All text that you want to display onscreen should be placed inside these brackets
  6. h1 through h6: Defines headings (h1 is the largest font size, and h6 is the smallest)
  7. p: Defines a paragraph (This is particularly placed under h1s and h6s)
